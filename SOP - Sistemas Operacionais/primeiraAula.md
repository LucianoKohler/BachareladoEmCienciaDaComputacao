Sistemas Operacionais - Rafael Obelheiro

## O que é um SO?
- Um software que serve para ser a "ponte" entre as aplicações utilizadas pelo usuário e o hardware que irá computar os comandos14

## Plano de aula
1. Fundamentos
2. Gerência de processador
    - procesos e threads
    - programação concorrente
    - escalonamento
3. Gerência de memória
4. Sistemas de arquivos
5. Gerência de E/S
6. Deadlocks (impasses)
7. Virtualização
