# Processos de Software

*Um processo define quem está fazendo o que, quando e como para atingir certo objetivo*

**Artefato**: Todo material que envolve o desenvolvimento (código, diagrama, logo, etc...)

## Modelo Cascata
Segue uma sequência de passos sequencial
- Comunicação
- Planejamento
- Modelagem
- Construção
- Implantação

Bom para organização, mas é difícil levantar TODOS os requisitos necessários no início, então voltas acontecem

## Cascata em V e W
Foco em desenvolver olhando pro futuro (testagem)
- Levantamento de requisitos -> Teste de aceitação
- Codificação -> Teste unitário 
- ...

# Modelos 
- Sequenciais
    - O prejeto é feito de uma vez só, do início ao fim
- Incrementais
    - Trabalha com ciclos, a cada ciclo, uma nova parte do projeto se junta ao produto final
- Evolutivos
    - Envolve técnicas mais desenvolvidas, mas se tornam complexos de se controlar
- Métodos Ágeis
